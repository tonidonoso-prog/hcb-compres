# Lector de PDF Simple

Este proyecto permite extraer texto de archivos PDF de forma masiva y automática.

## Estructura del Proyecto
- `input/`: Coloca aquí los archivos PDF que deseas leer.
- `output/`: Aquí se guardarán los archivos `.txt` con el contenido extraído.
- `main.py`: El script principal encargado de la lectura.

## Cómo usar
1. Asegúrate de tener Python instalado.
2. Instala la librería necesaria ejecutando:
   ```bash
   pip install -r requirements.txt
   ```
3. Coloca tus archivos PDF en la carpeta `input/`.
4. Ejecuta el script:
   ```bash
   python main.py
   ```
5. ¡Listo! Los archivos TXT aparecerán en la carpeta `output/`.
